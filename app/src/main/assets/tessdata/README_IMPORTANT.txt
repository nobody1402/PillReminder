این پوشه باید فایل fas.traineddata (داده‌ی زبان فارسی برای موتور OCR آفلاین Tesseract) رو داشته باشه.

چون این فایل حجمش چند مگابایته و در محیط ساخت این پروژه اینترنت در دسترس نبود، باید
یک‌بار خودتون دستی دانلودش کنید:

۱. برو به آدرس:
   https://github.com/tesseract-ocr/tessdata_fast
   (یا برای دقت بیشتر و حجم بیشتر: https://github.com/tesseract-ocr/tessdata)

۲. فایل fas.traineddata رو دانلود کن.

۳. اون رو دقیقاً توی همین مسیر بذار:
   app/src/main/assets/tessdata/fas.traineddata

۴. این فایل README رو می‌تونی حذف کنی.

بعد از این کار، بخش «افزودن دارو از روی عکس نسخه» به‌صورت کاملاً آفلاین (بدون اینترنت)
کار می‌کنه. اگه این فایل رو نذاری، اپ کرش نمی‌کنه؛ فقط توی همون صفحه پیام خطای واضح
نشون می‌ده که فایل زبان پیدا نشد.
